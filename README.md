## About me

I design and ship **AI-powered systems** end to end — from multi-provider agent frameworks and Model Context Protocol servers to the **cloud infrastructure** (Terraform on AWS, self-hosted Docker stacks) and **identity/SSO** plumbing that holds it all together. I like small, modular, reusable building blocks that compose into something genuinely useful, and I'm equally at home in Python, .NET, and Java.
